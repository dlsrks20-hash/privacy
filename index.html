개인정보처리방침

시행일: 2026년 3월 23일

1. 수집하는 개인정보

본 앱("우리의 계절")은 서비스 제공을 위해 다음 정보를 수집합니다:

  • 이메일 주소 (회원가입 및 로그인)
  • 프로필 정보 (닉네임, 성별, 프로필 사진)
  • Google 계정 정보 (Google 로그인 시: 이름, 프로필 사진 URL)
  • 위치 정보 (파트너/그룹 위치 공유 기능 사용 시)
  • 일기 내용 및 첨부 사진
  • 그룹 연동 정보 (그룹명, 멤버 구성)

2. 개인정보의 이용 목적

  • 회원 식별 및 인증
  • 파트너 연동 서비스 제공
  • 일정 관리 및 일기 기록 서비스 제공
  • 위치 기반 서비스 제공 (동의 시)

3. 개인정보의 보관 및 파기

회원 탈퇴 시 모든 개인정보는 즉시 파기됩니다. 관련 법령에 따라 보관이 필요한 정보는 해당 기간 동안 보관 후 파기합니다.

4. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 법률에 의해 요구되는 경우는 예외로 합니다.

5. 위치 정보

위치 정보는 사용자가 명시적으로 "내 위치 전송" 버튼을 누를 때만 수집되며, 연동된 파트너에게만 공유됩니다. 위치 정보는 서버에 저장되며 연동 해제 시 삭제됩니다.

6. 데이터 보안

모든 데이터는 SSL/TLS 암호화를 통해 전송되며, Supabase 클라우드 서비스를 통해 안전하게 저장됩니다. Row Level Security(RLS) 정책을 적용하여 본인의 데이터만 접근 가능하도록 보호합니다.

7. 사용자 권리

  • 개인정보 열람, 수정, 삭제 요청
  • 위치 정보 수집 동의 철회
  • 회원 탈퇴
